# 검사항목 마스터 데이터 v3

미모 수의 검사 데이터 관리 시스템용.
정규항목 106개 + Alias 60개 = 총 166개 이름 인식 가능.

---

## 항목 매핑 로직

검사 결과 입력 시 항목명을 자동으로 정규 항목에 연결하는 4단계 프로세스:

```
입력: 검사지의 항목명 (예: "Hct(ABL80F)", "ALP", "vLIP")

Step 1. 정규 항목과 exact match (case-insensitive)
        → 매칭되면 바로 저장
        ↓ 실패
Step 2. Alias 테이블에서 exact match (case-insensitive)
        → 매칭되면 canonical 항목으로 저장 + source_hint 기록
        ↓ 실패
Step 3. AI 판단
        → 기존 항목의 변형인가? → Yes: canonical에 매핑 + 새 alias 자동 등록
        → 완전 신규 항목인가?   → Yes: Step 4로
        ↓
Step 4. 신규 항목 자동 분류
        AI가 추천한 exam_type, organ_tags 기반으로
        → test_items에 신규 정규 항목 추가
        → 4가지 정렬 체계에 자동 배치
```

### Step 3 AI 판단용 프롬프트 템플릿

```
다음 검사 항목명이 기존 정규 항목 중 하나와 같은 검사인지, 아니면 신규 항목인지 판단해줘.

입력된 항목명: {input_name} (단위: {input_unit})

기존 정규 항목 목록:
{canonical_list}

판단:
- match: 기존 항목과 동일하면 → canonical_name 반환
- new: 신규 항목이면 → 추천 정규명, display_name_ko, exam_type, organ_tags 반환
```

### 검사 결과 기록 저장 구조

```json
{
  "record_id": "REC_001",
  "item_id": "ITEM_016 (→ HCT)",
  "value": 42.5,
  "date": "2025-01-15",
  "source_name": "Hct(ABL80F)",
  "source_hint": "ABL80F",
  "hospital": "A동물병원"
}
```

→ 트렌드 차트는 `item_id`(HCT) 기준으로 통합 조회.
→ 개별 기록의 `source_name`으로 "이 값은 어떤 장비에서 나왔는지" 확인 가능.

## 4가지 정렬 체계

### 1. 검사유형별
장비/검사 방법 기준 (검사 결과지 순서)

순서: Vital → CBC → Chemistry → Special → Blood Gas → Coagulation → 뇨검사 → 안과검사 → Echo

### 2. 장기/시스템별
신체 부위 기준 (하나의 항목이 여러 장기에 태그 가능)

분류: 기본신체, 혈액, 간, 신장, 췌장, 심장, 전해질, 산염기, 호흡, 지혈, 면역, 염증, 대사, 내분비, 근육, 뼈, 담도, 영양, 알레르기, 감염, 안과

### 3. 임상 우선순위별
응급/중환자 상황에서 확인 순서

- **1. 활력징후**: BT, Pulse, Systolic BP, BW
- **2. 산소/환기**: pO2, pO2(T), sO2, pCO2, pCO2(T) 외 1개
- **3. 산염기**: pH, pH(T), cHCO3, cHCO3(P,st), BE 외 9개
- **4. 전해질**: Na, K, Cl, Calcium, Phosphorus 외 1개
- **5. 빈혈/적혈구**: HCT, HGB, RBC, MCV, MCH 외 10개
- **6. 백혈구**: WBC, NEU, %NEU, LYM, %LYM 외 8개
- **7. 지혈/응고**: PLT, MPV, PDW, PCT, PT 외 7개
- **8. 간/담도**: ALT, AST, ALKP, GGT, T.Bilirubin 외 2개
- **9. 신장**: BUN, Creatinine, BUN:Cr Ratio, SDMA, UPC 외 3개
- **10. 단백질**: Protein-Total, Albumin, Globulin, A/G ratio
- **11. 췌장/대사**: Glucose, Lipase, Amylase, cPL, Triglycerides 외 1개
- **12. 특수/심장**: CRP, CK, LDH, proBNP, 심장사상충 외 2개
- **13. 안과**: 눈물량(OD), 눈물량(OS), 안압(OD), 안압(OS)

### 4. 검사 패널별
병원에서 한번에 돌리는 검사 세트 기준

- **Basic** (기본 혈액검사): 10개 항목
- **Pre-anesthetic** (마취 전 검사): 14개 항목
- **Senior** (노령견 종합): 26개 항목
- **Pancreatitis** (췌장염 집중): 8개 항목
- **Coagulation** (응고 검사): 9개 항목
- **Emergency** (응급/중환자): 13개 항목
- **Cardiac** (심장 검사): 6개 항목
- **Kidney** (신장 집중): 13개 항목
